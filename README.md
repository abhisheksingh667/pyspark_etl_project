# PySpark ETL Pipeline Project

## 🚀 Overview

Built an end-to-end ETL pipeline using PySpark and Delta Lake on Databricks.

## 🔧 Tech Stack

* PySpark
* Delta Lake
* Databricks
* Python

## ⚙️ Pipeline Steps

1. Data Extraction from source table
2. Data Cleaning (null handling, duplicates removal)
3. Data Transformation (date conversion, feature engineering)
4. Aggregation (region-wise, monthly sales)
5. Data Loading into Delta Tables
6. Optimization using partitioning

## 📊 Output Tables

* clean_sales_train
* region_sales_summary
* sales_partitioned

## 💡 Key Features

* Scalable data processing using PySpark
* Efficient storage using Delta Lake
* Partitioning for performance optimization
